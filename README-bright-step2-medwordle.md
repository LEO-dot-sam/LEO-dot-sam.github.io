# Bright Step 2 MedWordle Update

Replace your current MedWordle files with:

- `medwordle.html`
- `medwordle.css`
- `medwordle.js`

Upload them to the root of your GitHub Pages repository and commit changes.

Live page:
`https://leo-dot-sam.github.io/medwordle.html`

## New features

- Bright/light UI
- Specialty color themes
- Specialty selector
- Mode selector:
  - High Yield
  - All Outline
  - Hard Mode
- NBME-style hints
- Clinical pearls after correct answer or after the game ends
- Saved user preferences with localStorage
