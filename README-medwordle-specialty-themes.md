# MedWordle Specialty Themes Update

Replace your old MedWordle files with these three files in the root of your GitHub Pages repo:

- `medwordle.html`
- `medwordle.css`
- `medwordle.js`

Then commit changes.

Your page will be live at:

`https://leo-dot-sam.github.io/medwordle.html`

This version includes:
- specialty selector
- specialty-specific colors
- All Medicine random mode
- saved specialty preference using localStorage
