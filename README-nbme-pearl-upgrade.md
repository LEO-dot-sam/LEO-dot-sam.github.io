# MedWordle NBME-Style Pearl Upgrade

Replace your current MedWordle files with:

- `medwordle.html`
- `medwordle.css`
- `medwordle.js`

## New pearl format

After the game ends, users see:

- Key Clue
- Most Likely Diagnosis
- Initial Management
- Next Best Step
- NBME Takeaway

Daily Challenge remains universal and locked:
- Same word for everyone each day
- Specialty and difficulty ignored
- Pearl appears after win or loss
