# Lionel Sam Research Portfolio

Static GitHub Pages website for Lionel Sam.

## Files
- `index.html` — main website content
- `style.css` — styling and responsive design
- `script.js` — small footer year script
- `Curriculum_Vitae_LSAM.pdf` — optional CV file for the Download CV button

## How to publish on GitHub Pages

1. Go to GitHub and create a new repository named:

   `YOURUSERNAME.github.io`

2. Upload these files into the repository:

   - `index.html`
   - `style.css`
   - `script.js`
   - optional: `Curriculum_Vitae_LSAM.pdf`

3. Commit the files.

4. Go to **Settings → Pages**.

5. Under **Build and deployment**, set:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

6. Save.

7. Your site should appear at:

   `https://YOURUSERNAME.github.io`

## Important edits

In `index.html`, replace any placeholder details you want to change, especially the CV filename if needed.


## Updating publications
Replace the files in the repository root with these updated files, then commit changes. Publication links open in new tabs.


## Image assets
This version includes an `images/` folder with custom SVG academic mark cards and stylized book-cover cards. Upload the entire folder along with `index.html`, `style.css`, and `script.js`.
