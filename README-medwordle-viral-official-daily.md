# MedWordle Viral + Official Daily Update

Replace your current MedWordle files with:

- `medwordle.html`
- `medwordle.css`
- `medwordle.js`

Upload to the root of your GitHub Pages repo and commit.

## What changed

- Daily Challenge is now universal and restricted:
  - Same word for everyone each day
  - Ignores specialty and difficulty
  - Locks after completion on that device/browser
- Daily Challenge has an official banner
- Share button now creates a Wordle-style emoji grid
- Streak appears in the share result
- Practice Weak Topic button appears after Daily Challenge
- Practice and Adaptive modes still allow unlimited studying
- White UI is preserved
